# Fitness Coach Analytics Pipeline

End-to-end analytics pipeline für Fitness-Coaches — von Rohdaten bis zum interaktiven Dashboard.

## Was dieses Projekt zeigt
- Python Pipeline die Daten analysiert und in Google Sheets lädt
- KPIs: Aktive Klienten, MRR, Churn Rate, Umsatz pro Paket
- Interaktives Looker Studio Dashboard mit 3 Seiten

## Live Dashboard
https://lookerstudio.google.com/reporting/dd59615f-0755-4453-9406-625b307797e2

## Tech Stack
- Python (pandas, gspread)
- Google Sheets API
- Looker Studio

## Projektstruktur
- `fitness_analysis.py` — Haupt-Pipeline
- `fitness_coach_data.csv` — Simulierter Datensatz (50 Klienten)
- `SETUP_GUIDE.md` — Setup Anleitung

## Datensatz
- 50 simulierte Fitness-Coach Klienten
- KI-generierte Beispieldaten (kein echter Kunde)
- Felder: Klient ID, Name, Paket, Preis, Start/Enddatum, Status, Check-ins, Umsatz
